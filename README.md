# Supervised Machine Learning Models Portfolio

## Overview
Comprehensive collection of supervised learning projects featuring regression and classification models.

## Projects Included
1. **Regression & Classification - TITANIC**: Predictive models on Titanic dataset
2. **Cerebral Stroke Prediction**: Medical classification using health indicators
3. **Decision Tree - Titanic**: Tree-based classification approach

- Decision Trees
- Random Forests
- Feature Engineering
- Model Evaluation (Precision, Recall, F1-Score)

- Jupyter Notebooks
